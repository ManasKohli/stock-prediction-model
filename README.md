# NVIDIA Stock Price Prediction with LSTM Neural Networks

A machine learning project that uses Long Short-Term Memory (LSTM) neural networks to predict NVIDIA stock prices. This project leverages TensorFlow's Sequential API to build and train a deep learning model capable of forecasting stock price movements based on historical data.

## üß† Model Architecture

The model uses a Sequential architecture with LSTM layers:

- **Input Layer**: Time-series sequences of stock price data
- **LSTM Layers**: One or more LSTM layers with varying units to capture temporal patterns
- **Dropout Layers**: Regularization to prevent overfitting
- **Dense Layer**: Output layer for price predictions

### Typical Model Structure:
```python
model = Sequential([
    LSTM(50, return_sequences=True, input_shape=(time_steps, features)),
    Dropout(0.2),
    LSTM(50, return_sequences=False),
    Dropout(0.2),
    Dense(1)
])
```
